### 🐛 关键修复 (Critical Fixes)

- **PDF 导出黑框修复**:
  - **修复文字黑边问题**: 彻底解决了导出 PDF 时，每个文字周围出现黑色描边的问题。原因是 Tauri WebView 在通过 `dom-to-image` 渲染文字到 Canvas 时，错误添加了 `text-stroke` 样式。现已在导出过程中强制移除。

- **PDF 分页切割修复**:
  - **修复文字被切断问题**: 修复了多页 PDF 导出时，分页边界切到文字中间的严重 Bug。之前使用硬编码的近似格子大小（96px），现已改为从 DOM 动态读取实际网格配置（gap、cellStride），确保永远在格子边界切割。
